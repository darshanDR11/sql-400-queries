# SQL 400 Queries Project

This repository contains 400 SQL practice queries organized across multiple real-world databases.

## Databases Used

1. Sakila Movie Rental Database
2. E-commerce Database
3. University Database
4. HR Management Database

Each database contains:
- Schema
- Problem Statements
- SQL Solutions

## SQL Concepts Covered

- SELECT queries
- Filtering
- GROUP BY & HAVING
- JOINS
- Subqueries
- CTE (Common Table Expressions)
- Window Functions
- RANK / PARTITION
